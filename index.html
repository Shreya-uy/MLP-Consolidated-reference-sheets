This page contains links to pages containing a summary of sklearn APIs covered in the lectures

The sklearn APIs covered in the lectures are summarised under 4 categories
    - Data generation/preprocessing & model selection
    - models
    - evaluation
    - otherrelatedAPIs - These are APIs needed to understand other/APIs or referred to infrequently in the lectures
For sklearn API details, please use ?help followed by API name on google colab. For common regression & classification API names replace the term regression/regressor by classification/classif (Indicated in the sheet).
